## 🚀 Core Capabilities
Optimized for the **Azure AI stack** and **MLOps** lifecycles:

*   **🔍 Hybrid Retrieval:** Advanced Semantic + Keyword search using Azure AI Search vector indexing.
*   **🛡️ Enterprise Security:** Native Azure Entra ID (RBAC) integration. No exposed secrets.
*   **📡 Observability:** Integrated with Azure Monitor for real-time performance tracking and tracing.
*   **🌹 Vision Project Support:** Specialized context for the **Automated Rose Counting** project.

## ⚖️ Real-time Quality Audit (LLM-as-a-Judge)
Every interaction is audited by a secondary AI agent to ensure technical accuracy:
1.  **Faithfulness:** Verifies if the response is strictly grounded in the retrieved documents.
2.  **Relevance:** Measures how effectively the query was addressed.
*Check the side panel (Metrics) to see the audit results for each message.*
