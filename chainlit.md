# 🤖 Fuyu Intelligence v4.0
### *Enterprise RAG Solution & MLOps Tutor*

Bienvenido al ecosistema **Fuyu Tech**. Este asistente no solo responde preguntas, sino que opera bajo una arquitectura de **Confianza Cero (Zero-Trust)** y se auto-audita en tiempo real para garantizar la veracidad de la información.

---

## 🚀 Capacidades del Sistema
Este tutor está especializado en el stack tecnológico de **Azure AI** y flujos de trabajo **MLOps**, con un enfoque particular en:
* **Arquitecturas RAG:** Recuperación de información fáctica mediante búsqueda híbrida.
* **Seguridad Cloud:** Implementación de Managed Identities y RBAC (sin API Keys expuestas).
* **Visión Artificial:** Soporte especializado para el proyecto de **Conteo Automatizado de Rosas**.

## ⚖️ Auditoría de IA Integrada
Cada respuesta generada pasa por un **Juez Auditor** que califica:
1.  **Fidelidad:** ¿La respuesta se basa estrictamente en las fuentes?
2.  **Relevancia:** ¿Resuelve realmente la duda del usuario?
*Podrás ver estas métricas en el panel lateral de cada mensaje.*

## 🛠️ Cómo interactuar
* **Haz preguntas técnicas:** "¿Cómo asigno el rol de Search Index Data Reader?"
* **Consulta sobre el proyecto de Rosas:** "Explícame el flujo de procesamiento de video para el conteo."
* **Cita de fuentes:** El bot siempre te indicará de qué documento extrajo la respuesta (Fuente X).
